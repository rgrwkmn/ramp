Social Flyer JS

adf.js - Application framework
script.js - Bootstrap
script-built.js - Combined jquery, plugins, modules framework and main script
script-built.min.js - Same as above but minified
plugins.js - Framework level plugins
plugins/ - Module level plugins
modules/ - Framework Modules
libs/ - Framework libraries (jQuery, RequireJS)

jsdoc-toolkit/ - Auto-docs from code comments
docs/ - Generated reference documentation
generatedocs.sh - Run this from a shell to generate docs from script.js, adf.js and modules/
